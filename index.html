Para la suma, cuando pones un numero con un signo "+" y otro numero, el computador reconoce que quieres juntar dos elementos, no sumarlos.

Es por eso que cuando pones este codigo

if (pregunta == "sumar"){
alert("la respuesta es " + (numero1 + numero2))
}

dos numeros se juntan, entonces tenemos que 2 + 5 no es 7 sino 25
